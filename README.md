# MES-LuaDoc

This package created for generate documentation for Lua language. This package is meant to be one movement to simplify your life.

Create *js* file with similar code

```javascript
const MesLuaDoc = require('mes-luadoc');

MesLuaDoc.MesLuaDoc('.')
```
